Macro Description
General macros
$PlatformName$ Name of the platform
$UserName$ Name of the user
$UserID$ User ID
$Title$ Post title
$PageID$ Page ID
$ImagesCount$ Total number of images
$AttsCount$ Total number of attachments
Image macros
$ImageCounter$ Image number without leading zeros
$ImageCounter#X$ Image number with leading zeros
Replace X with the desired number of digits
$ImageName$ Original image name
Attachment macros
$AttCounter$ Attachment number without leading zeros
$AttCounter#X$ Attachment number with leading zeros
Replace X with the desired number of digits
$AttName$ Original attachment name
Publication time macros
$YYYY$ Year
$YY$ Year (short)
$MM$ Month
$DD$ Day
$hh$ Hour
$mm$ Minute
Current time macros
$NYYYY$ Year
$NYY$ Year (short)
$NMM$ Month
$NDD$ Day
$Nhh$ Hour
$Nmm$ Minute
